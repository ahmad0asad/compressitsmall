Checklists That Help You Pass Upload Limits (Fast)

Use these checklists for scholarships, visas, university portals, HR systems, and online forms. They help you avoid common rejections: “file too large”, “wrong format”, or “blurry scan”.

PDF upload checklist

  1. Confirm the max size and accepted formats.
  2. Extract only required pages: Extract Pages.
  3. Fix order: Reorder Pages.
  4. Compress last: PDF tools.
  5. Verify: Upload Limit Checker.

Image upload checklist

  1. Crop to the required ratio: Image Cropper.
  2. Export as JPG/PNG depending on content.
  3. Optimize if needed: Image tools.

Extra: Submission QA Checklist (before you click Upload)

  • File name: use simple names (letters/numbers), avoid special characters.
  • Readability: zoom to 200% and confirm text/signatures are readable.
  • Orientation: rotate scanned pages if needed.
  • Order: cover page → IDs → supporting docs → annexes.
  • Final size: confirm with Upload Limit Checker.

If you are repeatedly blocked by a strict limit, read How to compress a PDF and File size checklist.

Email & Messaging Checklist (attachments that actually send)

  • Gmail/Outlook limits: if you’re close to the cap, compress or extract pages first.
  • Prefer PDF for documents: one file is easier than multiple images.
  • For photos: JPG is usually smaller than PNG.
  • Rename files: avoid very long names and special symbols that some systems reject.

See: Email attachment limits.

Scan Quality Checklist (clear text without huge files)

  • Resolution: 200–300 DPI is usually enough for documents.
  • Color mode: grayscale for text documents; color only when necessary.
  • Remove blank pages: use Delete Pages.
  • Crop margins: excessive margins waste space and increase size.
  • Final step: compress only after cleaning and ordering.
