Upload Limit Checker: Instantly See If Your File Meets Portal Size Requirements

Nothing is more frustrating than spending time completing a long application form, attaching your documents carefully, clicking Submit — and then getting a vague file size error with no clear explanation of what to do next. This upload limit checker solves that problem before it happens. Drop any file to see its exact size in KB and MB, compare it against real-world portal limits, and get a direct link to the right tool for reducing it.

The checker runs entirely in your browser. Your file never leaves your device — only the file size metadata is read, not the content. This makes it safe to use with sensitive documents like passports, medical records, financial statements, or legal agreements.

How to use this tool

  1. Drop your file into the checker area, or tap to select it. Any file type works — PDF, JPG, PNG, DOCX, PPTX, XLSX, ZIP, and others.
  2. Your file size appears instantly in KB and MB. No waiting, no processing.
  3. Choose your target limit from the dropdown — 2 MB for scholarship or visa portals, 5 MB or 10 MB for most job portals, 25 MB for email attachments.
  4. See exactly how much smaller your file needs to be — both in absolute terms (e.g. "reduce by 1.4 MB") and as a percentage.
  5. Follow the recommended plan — the tool identifies your file type and links you directly to the most appropriate compression or splitting tool.

Why upload limits vary so much between portals

Upload limits are set by the platform developers and reflect server storage costs, processing capacity, and the typical file sizes they expect. A government immigration portal handling passport scans has very different requirements from a university submission system accepting research PDFs or a job board receiving CV documents.

The most common limits you will encounter in practice are 2 MB for visa and embassy applications, 5 MB or 10 MB for most university and scholarship portals, and 25 MB for standard email attachments. Some older government forms still enforce limits as low as 200 KB or 500 KB — particularly for photograph uploads on identity documents. Knowing the target limit before you start compressing saves you from having to repeat the process.

Common upload size limits by platform type

  • University and scholarship portals: 2 MB, 5 MB, or 10 MB per file. Varies significantly by institution — always check the specific portal instructions.
  • Visa and embassy applications: 1 MB or 2 MB per document. These limits are strictly enforced with no manual override.
  • Job application portals (LinkedIn, Workday, Taleo): typically 2 MB to 10 MB for CVs and cover letters.
  • Gmail attachment: 25 MB total per message. Files over 25 MB are automatically converted to Google Drive links.
  • WhatsApp document: 100 MB per file.
  • Government portals (HMRC, NHS, UK Home Office, DVLA): typically 2 MB to 10 MB depending on the document type.
  • Photo ID uploads (passport photos, profile images): often 200 KB to 1 MB with specific dimension requirements.

Why your file might be rejected even when it looks small enough

File size displayed on your computer and file size measured by a portal are not always the same number. There are a few reasons this happens.

First, Windows and macOS often display file sizes using decimal megabytes (1 MB = 1,000,000 bytes), while many portal validators use binary megabytes (1 MB = 1,048,576 bytes). A file showing 9.9 MB in Windows can exceed a portal's 10 MB limit because of this difference. Keeping your file at least 5–10% under the stated limit avoids this problem entirely.

Second, scanned PDFs often appear deceptively small in file browsers but contain high-resolution images on every page. The visible text is actually stored as image data, which is much heavier than vector text from a word processor. A 10-page scanned form at 300 DPI in colour can easily exceed 15 MB even though it contains only a few hundred words of text.

Third, Word and PowerPoint files may embed the original full-resolution version of images you resized inside the document. The image appears smaller on the slide or page, but the file still carries the original data. Compressing images inside the application directly — or exporting to PDF — usually resolves this.

Choosing the right approach for your file type

The most effective reduction method depends on what kind of file you have. For digital PDFs (created from Word, Google Docs, or similar), running the file through a PDF compressor usually reduces size by 20–50% without any visible quality loss. For scanned PDFs, the approach is different — switch to grayscale, reduce DPI to 150–200, and remove blank or duplicate pages before compressing. For images, converting from PNG to JPG and reducing dimensions to fit the display context typically gives the biggest size reduction. For Word and PowerPoint files, the main culprit is almost always embedded images — compress them inside the application or convert to PDF for submission.

Frequently asked questions

Does this tool upload my file to a server?
No. The file size is read entirely by your browser using the File API. Your file never leaves your device at any point. This tool reads only the byte count of the file — it does not open, parse, or transmit the file content.
Why do portals reject files that are close to the stated limit?
Binary vs. decimal measurement differences mean a file can appear under the limit on your computer but exceed it on the server. Additionally, some portals apply the limit to the encoded form of the upload, which can add overhead. Aim to stay 5–10% below the stated limit to avoid borderline rejections.
What file types work with this checker?
Any file — PDF, JPG, PNG, WEBP, HEIC, DOCX, PPTX, XLSX, ZIP, MP4, and anything else. The checker reads only the file size, so the file type makes no difference to how it works.
My file is already under the limit but the portal still rejects it — why?
Portal rejections can also happen because of file structure issues (encrypted PDFs, unusual internal formats), incorrect file type (a DOCX renamed to PDF), or dimension requirements for images (some portals require specific pixel dimensions, not just file size). If size is not the issue, try re-exporting or converting the file from scratch.
What is the fastest way to get a PDF under 2 MB for a visa or scholarship application?
Remove any blank or unnecessary pages first, then run one compression pass. If the PDF contains scanned pages, switching to grayscale at 150 DPI typically brings a 10-page document well under 2 MB. See our full guide on getting PDFs under 2 MB for a step-by-step walkthrough.
