DOCX to PDF (text-based conversion)

Extracts raw text from a DOCX file and places it into a PDF for simple submissions — cover letters, short reports and basic forms. No upload required.

DOCX → PDF
Open tool →

How it works

A DOCX file is a compressed ZIP containing XML. This converter reads word/document.xml, strips the markup, reconstructs clean text, and places it into a PDF using pdf-lib — entirely in your browser.

Best for

Quick PDF conversions where exact visual fidelity doesn't matter — administrative forms, cover letters, simple notices. For complex layouts with tables, columns or images, export from a full word processor instead.

Privacy

No server involved. Your file never leaves your device. The PDF is composed in browser memory and downloaded directly to your machine.

How it works

Paste your JSON into the input area and the formatter parses it with the browser's native JSON.parse(). If the structure is valid, it outputs a cleanly indented version. If there's an error, it tells you where the problem is.

JSON to CSV

Convert a JSON array of objects into a CSV file ready for Excel, Google Sheets or database imports. Keys become column headers automatically.

JSON → CSV
Open tool →

How it works

Paste a JSON array (e.g. from an API response or database export). The tool reads the keys from the first object to generate column headers, then maps every row into comma-separated values. Nested objects are flattened with dot notation.

Best for

Turning API responses, MongoDB exports and NoSQL data into spreadsheet-ready format. Supports semicolon delimiter for European CSV formatting.

Convert a CSV file or pasted CSV text into a JSON array of objects. Each row becomes an object with column header keys.

Open tool →

Upload a .csv file or paste CSV text. The first row is treated as headers. Each subsequent row becomes a JSON object with those headers as keys. The output is a valid JSON array ready to copy or download.

Text cleaner for office content

Paste text from DOCX or PDF, remove broken line breaks, normalise spacing, and straighten punctuation before reusing the content in forms, email or new documents.

Text clean
Open tool →

Why pasted office text looks broken

Copying from PDFs and DOCX files drags along invisible formatting — hard line breaks mid-sentence, multiple spaces, inconsistent quotation marks. The cleaner applies predictable transformations to fix this.

What it does

  • Removes leading/trailing whitespace per line.
  • Joins broken lines back into paragraphs.
  • Collapses repeated spaces to single spaces.
  • Optionally straightens curly quotes to plain ASCII.

Each option is a separate checkbox so you only apply what your content needs. Preview the output before downloading.

DOCX metadata remover (privacy cleaner)

Scrubs author names, revision timestamps and related fields from DOCX files by rewriting metadata XML inside the document package — all in your browser.

Privacy
Open tool →

Why DOCX metadata matters

Word files often embed the original author's name, organisation, creation timestamps, revision counts and editing duration. When shared externally, this metadata can reveal information you didn't intend to disclose.

What it removes

  • Author and "last modified by" names from docProps/core.xml.
  • Creation and modification timestamps.
  • Revision counts and editing duration where present.

Limits

This tool clears DOCX package metadata only — it doesn't scan the body text for names or email addresses written in the document itself. For full anonymisation, review the body text too.

Office file size analyser (DOCX / PPTX / XLSX)

Inspects the internal ZIP structure of DOCX, PPTX or XLSX files and shows which embedded parts consume the most space — images, media folders, unused layouts.

Analyse
Open tool →

Why office files get unexpectedly large

DOCX, PPTX and XLSX files are ZIP archives bundling XML with images, charts, embedded documents and style definitions. The culprit is usually not the text but high-resolution images, unused layouts or leftover media inside the package.

How to read the output

The report lists each internal file path with its uncompressed size in kilobytes, sorted largest first. Folders like ppt/media or word/media contain images. Items occupying hundreds of kilobytes are your primary optimisation targets.

What to do next

The analyser only reveals structure — it doesn't modify your file. Use the findings to guide clean-up: replace oversized images, delete unused slide masters, or strip embedded objects in Word or PowerPoint directly.
